This project predicts whether a person is diabetic or not using machine learning models based on medical attributes. It compares multiple algorithms and improves performance using SMOTE to handle class imbalance.

# Dataset
Kaggle Dataset

# Features include:
Age
BMI
HbA1c level
Blood glucose level
Hypertension
Smoking history

# Technologies Used:
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn

# Machine Learning Models:
Logistic Regression
Naive Bayes
Decision Tree
Random Forest
XGBoost

# Results
Best Model: XGBoost with SMOTE
Macro Recall Score: 0.88
Improved performance on imbalanced dataset
